Real Estate API

Real-ish API on Real Estate Data in Belgium - only for Testing and Prototyping
powered by JSON Server and lowdb

Intro

This uses JSONPlaceholder - a free online REST service that you can use whenever you need some fake data.

It's great for tutorials, faking a server, sharing code examples, ...

Data

Not all fields are available on all properties - be careful

urlOriginal URL
locationAddress (full or just municipality)
pricePrice (integer, in Euros)
descriptionFree text description
propertiesArray of various info
imageURL of image
CoordinatesGPS coordinates (lat, lng)
